# Auftragstool

## 🧾 Beschreibung
Webbasierte Anwendung zur Erfassung und Verwaltung von Arbeitszeiten sowie deren Zuordnung zu Projekten.

Ziel ist es, ein effizientes und sauberes Controlling-Tool für den Einsatz im Unternehmen bereitzustellen.

---

## ⚙️ Features
- Arbeitszeiten erfassen (inkl. Pausenlogik)
- Excel-Import für Arbeitszeiten
- Projektzuordnung von Stunden
- Rollen- und Login-System
- KPI- und Monitoring-Dashboard
- Export-Funktionen

---

## 🧠 Tech Stack
- Next.js
- TypeScript
- Supabase
- Tailwind CSS

---
